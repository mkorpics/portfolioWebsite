Too Fantastic Fashion

Superhero wardrobe application

Task: Utilizing database-first skills, built a web application allowing users to manage their wardrobe, viewing items by clothing type.

Skills: Database-first MVC, C#, SQL, HTML5, CSS3, Bootstrap

Through SQL Server, fashioned multiple tables linked through foreign key relationships. Generated an application that allowed users to manage both individual articles and also outfit collections. Modified the articles controller to allow each article type to have its own create, edit, details and delete pages. Through the outfits controller, filtered the first few drop down menus to display only particular clothing types.

With the film The Incredibles as general inspiration, I built my website around an ironic, tongue-in-cheek superhero theme. Set in the world of animated (as opposed to live-action) superheroes, the site encourages large personality. It allows users to create their own colors, occasions and seasons but comes equipped with a basic array of cliche-oriented options. Ideally, it caters to a younger audience with active imaginations, who could act as a superheroes themselves or create a page for their own favorite superhero!
